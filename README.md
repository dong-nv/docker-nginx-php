# sunasteriskrnd/nginx-php

Nginx image for php-fpm. Environment variables:

| Name          | Description   | Default                |
|---------------|---------------|------------------------|
| PHP_FPM_HOST  | php-fpm host  |                        |
| PHP_FPM_PORT  | php-fpm port  | `9000`                 |
| DOCUMENT_ROOT | Document root | `/var/www/html/public` |
